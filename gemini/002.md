## 🤖 프로젝트 보고서: 자유의 날개 (Project Freedom Wings)
    작성일: 2026-03-15
    작성자: 마술사 & Gemini

    1. 개요
    로봇 팔의 5번 관절(그리퍼)의 동작 불안정성 해결 및 소프트웨어-하드웨어 간 제어 정합성 최적화.

    2. 주요 해결 이슈
    2.1 하드웨어 부하 및 열폭주 현상

        - 현상: 특정 각도 구간에서 모터가 웅웅거리며 발열 발생.

        - 원인: 물리적 기구부와 소프트웨어 명령값 간의 오차로 인한 무리한 토크 발생.

        - 해결:
        5번 모터 교체 실시.

        90도 부근에서의 최대 겹침 포인트를 테스트하여 실제 물리적 한계점 확보.

        안전 범위 설정: 85도 이상에서 잡음이 발생하는 현상을 발견하고, 이를 제어 로직에 반영.
    

    3. 구현 내용
    3.1 펌웨어(Rust) 로직 보강

        안전 제어: 5번 모터 제어 시 0~85도 사이의 안전 구간을 설정하여 모터 보호 및 잡음 방지.

    3.2 iOS 제어 로직 (Swift)

        동적 UI 제어: UISliderView의 최대값을 모터 ID에 따라 동적으로 제한.

## Swift
```swift
// ContentView.swift 내 수정된 로직
maxValue: bleManager.selectedMotor == 5 ? 85.0 : 180.0
//데이터 클램핑: 전송 데이터가 범위를 벗어나지 않도록 방어 로직 추가.
```

    4. 학습된 공학적 교훈
        조립 ≠ 정렬: 기구적 조립보다 중요한 것은 데이터와 물리적 위치를 일치시키는 보정(Calibration) 과정이다.

        안정성 제1원칙: "속도보다 안정성"을 위해 하드웨어의 무리한 구간을 과감히 컷오프하는 것이 장기적인 시스템 신뢰성을 높인다.

        인간-AI 파트너십: AI의 시뮬레이션 기반 추론과 인간의 현장감 있는 문제 해결(납땜, 기어 조정, 열 확인)이 결합할 때 최고의 퍼포먼스가 나온다.

    5. 향후 작업 계획

    [ ] 나머지 관절들에 대한 물리적 위치-펄스값 매핑(Calibration) 진행.

    [ ] 여러 관절을 동시에 움직이는 '순차 제어(Sequencing)' 테스트.

    [ ] 최종 사과 깎기 알고리즘 구현.
